# Quickstart: GitHub Pages 静态单页站点

**Branch**: `005-pages-static-spa`  
**Created**: 2026-04-20

## Local

- Install deps: `npm install`
- Run Web dev server: `npm run dev:web` then open `http://127.0.0.1:5173/`
- Run checks: `npm run lint` / `npm test` / `npm run build`

## GitHub Pages

- Goal: after merging to `main`, the site is built and deployed automatically.
- Expected outcome: opening the GitHub Pages URL loads the app and remains functional on refresh/deep links.
