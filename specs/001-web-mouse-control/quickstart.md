# Quickstart: dev:web 鼠标操作支持

## 1. 启动

```bash
npm install
npm run dev:web
```

打开 `http://127.0.0.1:5173/`。

## 2. 冒烟验证

1. 在页面上点击“开始对局”
2. 在球桌区域按下鼠标并拖动，观察瞄准/力度反馈
3. 释放鼠标，确认只触发一次出杆
4. 等待球停止，确认可再次出杆
5. 验证取消路径（例如 ESC/右键取消）不会触发击球
6. 验证重开与返回菜单可点击并生效

## 3. 边界验证

1. 球运动中拖拽鼠标，不应触发新出杆
2. 球桌外按下/释放，不应触发出杆
3. 快速连续点击，不应重复出杆
4. 极小/极大拖拽下，力度应被限制在 0~1

## 4. 质量门禁

```bash
npm run lint
npm test
npm run build
```
